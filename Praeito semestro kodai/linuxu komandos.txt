--piping--(vienos komandos outputa paduodi kitai komandai kaip inputa)
> locate (pvz.) csv | less -pipe locate into less( view things in less mode)
> locate csv | grep domain -pipe locate into grep ( we gonna see files names that have word "domain" in it)
> df -h | grep media - pipe df -h into grep(we gonna see system disk space usage only with word "media")
> cat jdthesis.txt | grep Jesus -that is gonna output lines that have word 'Jesus'
		same but better way to write it >grep Jesus jdthesis.txt
> cat jdthesis.txt | grep Jesus | less - same thing, but in more simpler way(look less com.)
> cat jdthesis.txt | tr W J  -gonna replace 'W' in the text with 'J' and output it.
> cat jdthesis.txt | tr "[:lower:]" "[:upper:]"  -this gonna replace all lower cases with with upper cases
----
--writing/creating new files--
> grep Jesus jdthesis.txt > jesus.log  -all my lines with 'Jesus' gonna be in new file jesus.log
> ls >> ls.log  -that is going to write ls into ls.log
	DIFERENCE BETWEEN '>>' and '>'  
		>> - if there is something in the file, new text going to be put beneath it( at the end of the file)
		>  - going to write new things at the front of the file  
----
